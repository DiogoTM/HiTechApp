### *Description*

position​ ​inside​ ​the​ ​company.​ ​It​ ​also​ ​allows​ ​the​ ​company​ ​to​ ​have​ ​a​ ​complete​ ​record​ ​of​ ​his​ ​inventory,
clients,​ ​and​ ​orders.
